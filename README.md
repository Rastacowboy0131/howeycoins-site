# HoweyCoins 2.0

A parody/meme coin landing page inspired by the classic Investor.gov HoweyCoins fake ICO warning site.

## Run locally

```bash
cd /Users/rasta/howeycoins-site
npm start
```

Then open:

```text
http://localhost:4173
```

## Edit

- `index.html` — page copy and sections
- `styles.css` — design, colors, layout
- `app.js` — mobile menu and countdown

- Tropical fake-ICO style hero
- Lore section referencing the original HoweyCoins warning
